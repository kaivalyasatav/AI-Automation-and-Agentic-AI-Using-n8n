# GitHub PR Automation using n8n

An intelligent **GitHub Pull Request Automation workflow** built using **n8n** to streamline repository management by automating pull request monitoring, notifications, and workflow actions. This project showcases practical DevOps automation using event-driven workflows. :contentReference[oaicite:0]{index=0}

## Features
- Automated GitHub Pull Request monitoring
- Event-driven workflow automation
- PR notifications and status tracking
- GitHub API integration
- Real-time webhook-based triggers
- Scalable DevOps automation workflows
- Reduced manual repository management effort

## Tech Stack
- **n8n** – Workflow automation platform
- **GitHub API**
- **Webhooks**
- **REST APIs**
- **JSON Workflow Automation**

## Project Workflow
This automation listens for GitHub pull request events, processes incoming webhook data, performs automated actions such as notifications or validations, and helps streamline repository collaboration workflows.
