# WhatsApp AI Agent using n8n

An AI-powered WhatsApp automation agent built using **n8n** and **Agentic AI workflows** to handle intelligent conversations and automate responses in real time.

## Features
- WhatsApp message automation
- AI-powered conversational responses
- Real-time webhook integration
- Low-code workflow automation with n8n
- API integration support
- Scalable agentic workflow design

## Tech Stack
- n8n
- OpenAI / Gemini / LLM APIs
- WhatsApp Cloud API / Twilio
- Webhooks / REST APIs
